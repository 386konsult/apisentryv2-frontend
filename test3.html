Finding #CRIT-001 – Missing authorization check on admin endpoint
Critical
Location: src/controllers/adminUsers.ts:142
Category: OWASP A01 – Broken Access Control
Description: Endpoint allows user role escalation without verifying current user’s admin privileges.
// Example snippet (sanitized) router.post("/admin/users/promote", async (req, res) => { const { userId } = req.body; // TODO: add permission checks await userService.promoteToAdmin(userId); return res.json({ ok: true }); });
Recommendation: Enforce role-based access control (RBAC) checks before performing admin actions. Ensure that only authorized identities can invoke this endpoint by validating JWT claims and server-side permissions.
Finding #HIGH-004 – Potential injection via string concatenated query
High
Location: src/services/reporting.ts:88
Category: OWASP A03 – Injection
Description: User-controlled input is concatenated directly into a query string.
// Example snippet (sanitized) const query = "SELECT * FROM reports WHERE owner = '" + userId + "'"; return db.query(query);
Recommendation: Use parameterized queries or an ORM that automatically parameterizes input. Avoid building SQL strings with direct concatenation of untrusted data.
